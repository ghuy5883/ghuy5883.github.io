SPR2021 - DVD DB Search

A search app for a database of DVD entries.

Posted under Programming on April 4, 2021

Project Description

A practice project made while learning PHP and MYSQL. The database contains information of a large range of DVDs and can be searched for the details of a particular DVD. Users can also add, delete, or edit entries in the database.
